# RMMV-MessageFaceSounds
Bring Undertale-style dialogue sounds to RPG Maker MV! Map unique SEs and typing speeds to character faces via general parameters. Includes {spd:N} and {se:X} tags for per-dialogue overrides. Features natural pitch randomization, adjustable speed per face, 'cut/overlap' SE modes, and input skipping. Give your characters a unique voice!
